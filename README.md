# arithmetic-calculator

**Arithmetic Calculator (Tkinter)**

A simple GUI-based arithmetic calculator built with Python and Tkinter.
It performs basic operations between two numbers (integers or floats).

**Features**
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Input validation with error messages
* Reset and Exit buttons
* Read-only result field

**Requirements**
* Python 3.x
* Tkinter (comes pre-installed with Python)

**How to Run**
python calculator.py

**How It Works**
1. Enter two numbers.
2. Enter one operation (+, -, *, or /).
3. Click Result to see the output.

**Notes**
*Division by zero shows "Undefined Result".
*Invalid inputs display an error message.
